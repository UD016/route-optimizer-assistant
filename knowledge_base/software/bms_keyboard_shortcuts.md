## Purpose

This document provides a quick reference for commonly used keyboard shortcuts in the Business Management System (BMS).

Using keyboard shortcuts can improve efficiency when navigating forms, creating work orders, managing customer records, and performing day-to-day coordinator tasks.

---

# Record Management

| Shortcut | Function | Description |
|----------|----------|-------------|
| **F10** | Save | Save the current record. |
| **F6** | Insert Record | Insert a new record after the current record. |
| **Shift + F6** | Remove Record | Delete the current record. |
| **F4** | Copy Record | Copy the previous record. |
| **F3** | Copy Field | Copy the value from the same field in the previous record. |

---

# Query Functions

| Shortcut | Function | Description |
|----------|----------|-------------|
| **F7** | Enter Query | Enter Query mode. Press twice to retrieve the previous query criteria. |
| **F8** | Execute Query | Execute the query using the entered criteria. |
| **Ctrl + Q** | Cancel Query | Cancel Query mode. |

---

# Navigation

| Shortcut | Function | Description |
|----------|----------|-------------|
| **Page Up** | Previous Record | Navigate to the previous record. |
| **Page Down** | Next Record | Navigate to the next record. |
| **Ctrl + Page Up** | Previous Block | Move to the previous block on the current form. |
| **Ctrl + Page Down** | Next Block | Move to the next block on the current form. |

---

# Data Entry & Lookup

| Shortcut | Function | Description |
|----------|----------|-------------|
| **Ctrl + E** | Edit Text | Open the text editor window for entering or reviewing long text fields. |
| **F9** | Display List | Open the available values list, lookup form, or calendar depending on the selected field. |

---

# Printing & Help

| Shortcut | Function | Description |
|----------|----------|-------------|
| **Shift + F8** | Print | Print the current screen using the default printer. |
| **F1** | Help | Open the BMS online help documentation. |

---

# Closing Forms

| Shortcut | Function | Description |
|----------|----------|-------------|
| **Ctrl + Q** | Close Form | Close the active form when not in Query mode. |

> **Note**
>
> `Ctrl + Q` has two functions:
>
> - Cancels **Query Mode** when a query is active.
> - Closes the active form during normal operation.

---

# Frequently Used Shortcuts

These shortcuts are the ones Service Coordinators are most likely to use throughout the day.

| Shortcut | Typical Use |
|----------|-------------|
| **F10** | Save changes to a Work Order, Unit, Site, or Customer record. |
| **F6** | Create a new Site, Unit, or other BMS record. |
| **F7** | Search for existing records. |
| **F8** | Execute the search. |
| **F9** | Open lookup lists for customers, units, models, and other selectable fields. |
| **Ctrl + E** | Edit long Complaint, Cause, Correction, or Remarks text. |

---

# Tips

- Save frequently using **F10** to avoid losing work.
- Use **F7** and **F8** together to quickly search for existing records.
- Use **F9** whenever a lookup list is available instead of manually typing values.
- Use **Ctrl + E** when editing long text fields to improve readability and reduce formatting errors.
