# Technician Profiles Library v2

This ZIP contains one Markdown file per technician, rewritten into a retrieval-friendly format.

Included files: 20 total, including a placeholder for Louis Lauzon because no source profile data was present in the uploaded document.

Source material: `tech_profiles.md`
Generation date: 2026-07-16

## File structure

Each file uses a consistent layout:
- Purpose
- Metadata
- Summary
- Primary Territories
- Service Capabilities
- Cummins Training & Qualifications
- Security Clearances
- Work Preferences
- Awards (when present)
- Not Suitable For (when present)
- Operational Notes
- Search Keywords

## Notes

The files intentionally avoid dispatch recommendations. They are factual reference profiles only.
